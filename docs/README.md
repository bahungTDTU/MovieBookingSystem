# 📁 Documentation

This directory contains project documentation, screenshots, and diagrams for the CineWorld Movie Booking System.

## Structure

```
docs/
├── screenshots/    # Screenshots for evidence and reports
│   ├── phase1/     # Phase 1: Git workflow, Linux setup
│   ├── phase2/     # Phase 2: Traditional deployment
│   └── phase3/     # Phase 3: Docker deployment
└── diagrams/       # Architecture diagrams, flowcharts
```

## Guidelines

- Name screenshots descriptively: `phase1-github-branch-protection.png`
- Use PNG format for screenshots, SVG/PNG for diagrams
- Include timestamps in filenames if documenting a sequence
